CS6343 - Cloud Development

Welcome to the course CS6343 - Cloud Development. This course is designed to equip you with the knowledge and hands-on skills needed to build, deploy, and manage scalable, secure, and efficient cloud-based solutions.

You'll learn to leverage leading cloud platforms like AWS and Azure, work with essential cloud services, automate workflows with GitHub Actions, and containerize applications using Docker.
